Заголовок:
Начальная страница.

Роль:
Пользователь.

Цель:
Запуск программы открывает браузер с начальной страницей, где находятся кнопки "New Theory" и "New Practice". При нажатии на эти кнопки текущий блок заменяется на другой соответствующий блок, в котором можно ввести абсолютный путь к файлу или нажать кнопку в правой части ввода, чтобы открыть диалог выбора папки. Также можно ввести название файла в отдельном поле (не должно быть пустым) и отдельно название заголовка, которое будет отображаться в браузере (если пустое, то используется название файла). После этого можно нажать кнопку "Create", чтобы перейти к редактированию, или "Cancel", чтобы отменить создание и перейти к предыдущему блоку. Нажатие кнопки "Create" переносит пользователя на страницу редактирования соответствующего файла. Изначально он содержит базовые элементы - шапку, боковую панель, главную часть. Для theory шаблон example_theory, а для practice - example_practice.

Польза:
Начальная страница, с которой пользователь может начать работу с инструментом.

Критерии Приёмки:
1. При запуске программы открывается браузер с начальной страницей, где расположены кнопки "New Theory" и "New Practice".
2. При нажатии на кнопку "New Theory" текущий блок заменяется на блок для создания новой теории.
3. В блоке "New Theory" можно ввести абсолютный путь к файлу или выбрать папку через диалог выбора, а также ввести название файла и заголовок.
4. В блоке "New Practice" можно ввести абсолютный путь к файлу или выбрать папку через диалог выбора, а также ввести название файла и заголовок.
5. В блоке "New Theory" если поле для ввода пути к файлу пустое, то при нажатии на кнопку "Create" происходит запоминание пути по умолчанию (".\projects\theory", где "." заменяется на абсолютный путь к корню проекта).
6. В блоке "New Practice" если поле для ввода пути к файлу пустое, то при нажатии на кнопку "Create" происходит запоминание пути по умолчанию (".\projects\practice", где "." заменяется на абсолютный путь к корню проекта).
7. Если название файла пустое, то при нажатии на кнопку "Create" появляется фидбэк о том, что название файла не может быть пустым.
8. В блоке "New Theory" при нажатии на кнопку "Create" создаётся html файл и происходит переход на страницу редактирования этого html файла. Файл заполняется базовым шаблоном example_theory - шапка, боковая панель, главная часть.
9. В блоке "New Practice" при нажатии на кнопку "Create" создаётся html файл и происходит переход на страницу редактирования этого html файла. Файл заполняется базовым шаблоном example_practice - шапка, боковая панель, главная часть.
